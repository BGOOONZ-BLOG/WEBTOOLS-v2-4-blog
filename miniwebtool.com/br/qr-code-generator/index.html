O Gerador de código QR é usado para gerar códigos QR usando a API de gráficos do Google.

O que é o QR Code?

Um código QR é um código de matriz (ou código de barras bidimensional) que pode ser digitalizado usando o telefone da câmera e um leitor de código QR.
